# AJN Cordova

Project ini adalah default untuk assets www ketika membuat project baru Cordova.

## Fitur
Project ini sudah include beberapa tool mendasar yang sering dibutuhkan:

* Framework7 1.6.5
* Framework7 Icons
* Font Awesome 4.7.0
* Material Icons
* Custom Class CSS untuk perataan, messages info, dll
* Action Bar
* Toast
